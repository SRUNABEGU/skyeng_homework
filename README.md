## Проект создан для выполнения домашней работы и практики в использовании [GitHub](https://github.com/)
### Описание
В проекте есть функциональности для работы с большими объемами данных транзакций,\
которые позволяют **сортировать** их по описанию, типу валют, **генерировать** и **маскировать**\
номера **карт** и **счетов**, интегрирован **API** для **конвертации** USD и EUR в RUB,\
созданы инструменты **логирования** для отслеживания выполнения функций.\
Реализована работа с JSON, CSV и XSLX файлами.
## Зависимости
Общие:\
Python >= 3.10\
pandas >= 2.3.2\
python-dotenv >= 1.1.1\
openpyxl >= 3.1.5\
Разработка:\
pytest >= 8.4.1\
requests >= 2.32.4

Линтеры:\
black >= 25.1.0\
flake8 >= 7.3.0\
mypy >= 1.17.1\
isort >= 6.0.1
### Для работы некоторых функций
Необходимо создать файл ```.env```в корне проекта\
и заполнить его по шаблону: ```.env.sample```
## Установка
Клонирование репозитория:\
```git clone https://github.com/SRUNABEGU/skyeng_homework.git cd skyeng_homework```\
\
Установка зависимостей:\
```poetry install```
### Тестирование
Проект покрыт тестами Pytest на 100%.
Для запуска тестов нужно вызвать в терминале команду
```pytest --cov-report term-missing --cov=src```
