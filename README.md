# Домашняя работа!
Проект создан для выполнения домашней работы и практики в использовании [GitHub](https://github.com/)
## Установка
клонировать репозиторий и запустить файл ```processing.py```
## Описание
В проекте есть функциональности для работы с большими объемами данных транзакций которые позволяют сортировать
их по описанию, типу валют, и генерировать номера карт
### Требования
Для установки и запуска проекта, необходим [Python](https://www.python.org/downloads/) v3.10+.
### Тестирование
Проект покрыт тестами Pytest на 100%.
Для запуска тестов нужно вызвать в терминале команду
```pytest --cov-report term-missing --cov=src```
